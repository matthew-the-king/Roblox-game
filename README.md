# Roblox-game
This project only includes my game file for Survive 3 nights at Matt's hotel!

Survive 3 Nights at Matt's hotel is a horror game where a player is forced to run a hotel due to the fact that they need a job.  

This game is still a work in progress.  The purpose is to preview what the future game could look like.  When you go to sleep, that is the end of the game(Each day is 5 minutes).  

To view the game files, use a code editor and look at this link(size is 2 GB):https://drive.google.com/file/d/1UM95GUd3sr11RYdBrjAoxzCQllyvJ5TL/view?usp=sharing
